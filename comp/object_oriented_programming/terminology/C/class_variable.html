Variable shared with all functions stored inside container. Access specifier: public, protected, private
Public statements: access uninhibited.
Protected statements: access uninhibited to iteself and derived classes.
Private statements: access inhibited to all instances except itself. 

      class Dockyard {
      private:
	int sails();
	int boats();
      public:
	int getSails();
	int getBoats();
      }
